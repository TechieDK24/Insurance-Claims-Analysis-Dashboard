# 📊 Insurance Claims Analysis Dashboard

## 📌 Project Overview

The **Insurance Claims Analysis Dashboard** is an end-to-end Business Intelligence project developed using **Power BI, SQL, DAX, and Power Query**. The project transforms raw insurance data into meaningful business insights through interactive dashboards and KPI reporting.

It enables users to monitor customer distribution, policy performance, premium collection, claim trends, and coverage distribution using dynamic visualizations and filters.

# 📈 Dashboard Features

- Total Customers

- Total Premium
- Average Claims

### Interactive Visualizations

- Customers by State

- Coverage Distribution

### Interactive Filters

- State
- Policy Type
- Coverage
- Gender
